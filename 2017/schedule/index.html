Workshop: Transformando layouts em código

Language: pt-BR Complexity: Beginner

Colocando a Mão na Massa! Transformando layouts em código.
Você é aquela pessoa que quando tem um editor aberto da sua frente não sabe nem por onde começar? Tem dificuldade em colocar na prática tudo que aprendeu na parte teórica? Relaxe, você está no lugar certo! Nesse Workshop você vai colocar a mão na massa e desenvolver junto com a gente uma página do zero.
Recomendações:

  • Se você não manja nada, recomendamos que comece com a Introdução ao Javascript.
  • Aqui nós vamos usar Javascript Puro, nada de frameworks nem libs.
  • Vamos falar apenas de front-end, ok?
Programação:
  • Apresentações da equipe.
  • Softwares e ferramentas que vamos utilizar.
  • Apresentação do Layout.
  • Pensando na Modularização.
  • Criando nossa estrutura de diretórios.
  • Revisão sobre o DOM.
  • Aprendendo sobre eventos.
  • Criando as partes do projeto.
  • Como usar libs quando precisar.
  • Considerações finais.
Estruturas que vamos construir:
  • Menu Responsivo
  • Uma modal
  • Um botão que muda o texto de uma div
  • Usando uma lib para slides ou scroll da página

Docker no desenvolvimento Web

Language: pt-BR Complexity: Intermediate

Cada vez mais vemos a comunidade de desenvolvimento de software falar mais e mais sobre como entregar software de maneira rápida e com qualidade, onde DevOps, Entrega e Integração Contínua fazem parte dessas discussões. Baseado nisso, Docker se tornou uma das palavras do momento. Depois do uso das máquinas virtuais, atualmente tem se falado muito da conteinerização das aplicações. A idéia é mostrar como usar docker em uma aplicação React com um servidor ExpressJS. Serão abordados temas de "mount volume", docker-compose, testes em containers, deploy com docker, práticas para dockerizar aplicações web. E a pergunta de 1 milhão de dólares para que serve esse Docker e quais as vantagens de usar ele. Entenda em como "dockerizar" uma aplicação pode ajudar no desenvolvimento e na entrega dela.

GraphQL beyond the limit

Language: en Complexity: Advanced

The presentation covers an introduction to the GraphQL query language, explaining its features and benefits. I will show how minimal investment you have to do to start up your GraphQL endpoint without having to rewrite your entire database infrastructure. The workshop includes live demonstration (live coding).

Please have your own laptop with npm / yarn, node, RethinkDB pre-installed.

Schedule.js

  1. What is GraphQL, why is it good?
  2. Running Graphiql and writing your first schema.
  3. Learning Sequelize and connecting to the RethinkDB database.
  4. Merging Sequelize with GraphQL.
  5. Writing Mutations.
